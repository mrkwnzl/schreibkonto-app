Schreibkonto.app

Schreiben mit der 100-Wörter-Methode

Kontakt

Schreibkonto.app und die 100-Wörter-Methode wurden von Marko Wenzel erdacht und erstellt. Er arbeitet als Koordinator für Peer Learning am Zentrum für Lehren und Lernen an der Universität Bieleld.

Fragen & Anregungen bitte bevorzugt über GitHub. Wenn du keinen GitHub-Account hast, geht es auch per Mail.

Datenschutz

Die App speichert deinen Stand und deine Einstellungen komplett lokal in deinem Browser. Es werden keine Daten an Marko Wenzel gesendet und es werden keine Cookies gespeichert.

Den Code der App kannst du auf GitHub betrachten.

Schreibkonto.app wird auf GitHub Pages gehostet. GitHub Pages speichert IP-Adressen, mit denen Schreibkonto.app aufgerufen wurde. Mehr Informationen findest du hier.
